# @univedge/locus-ui

Locus Agent 的共享 Vue 3 UI 组件库，为 Web 前端提供可复用的界面组件。

## 安装

monorepo 内部包，由 `pnpm-workspace.yaml` 管理：

```jsonc
// package.json
{
  "dependencies": {
    "@univedge/locus-ui": "workspace:*"
  }
}
```

## Peer Dependencies

- `vue` >= 3.5
- `@vueuse/core` >= 14.0
- `@tanstack/vue-virtual` >= 3.0
