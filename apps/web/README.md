# @univedge/locus-web

Locus Agent 的 Web 前端，基于 Vue 3 + Vite 构建的单页应用，提供对话聊天、富文本编辑、知识库管理等交互界面。

## 功能

- **AI 对话界面** — 流式消息展示、工具调用可视化、上下文用量指示
- **富文本编辑器** — 基于 ProseMirror 的笔记编辑器，支持 Markdown
- **知识库管理** — 笔记、标签、记忆组织与语义搜索
- **看板视图** — 任务管理与进度追踪
- **代码变更追踪** — 查看 Agent 对工作区的文件变更

## 开发

```bash
# 在 monorepo 根目录
pnpm dev:web
```
